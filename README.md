Инструкция по использованию:
* Собрать проект
* Запустить проект
* Перед использованием методов апи необходимо выполнить запрос 

```bash
curl -X 'POST' \
  'https://localhost:5001/api/Tools/FillData' \
  -H 'accept: */*' \
  -d ''
```
